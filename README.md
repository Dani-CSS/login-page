# login-page

## Description
Project generated with Angular CLI 17, implementing a login and registration page.

## Features
- Login
- Register new user

## Technologies Used
- Angular CLI 17
- Typescript
- Sass

## Installation
1. Clone the repository
    ```bash
    git clone https://github.com/Dani-CSS/login-page
    ```
2. Navigate to the project directory
    ```bash
    cd login-page
    ```
3. Install the dependencies
    ```bash
    npm install
    ```
4. Run the project
    ```bash
    ng serve
    ```
